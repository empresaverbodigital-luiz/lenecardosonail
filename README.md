# 💅 Agenda Lene Cardoso Nail Designer

Organize seus atendimentos e controle seus recebimentos em um só lugar.

App web (PWA) completo: agenda com visão dia/semana/mês, cadastro de clientes, serviços, controle financeiro, contas a receber com cobrança via WhatsApp, relatórios, metas, lembretes, modo escuro, backup e instalação na tela inicial do celular.

## 🚀 Como publicar no GitHub Pages

1. Crie um repositório no GitHub (ex.: `agenda-lene`).
2. Envie **todos os arquivos desta pasta** para o repositório (index.html, app.js, manifest.json, sw.js, icon-192.png, icon-512.png). Pode arrastar os arquivos direto na página do repositório em **Add file → Upload files**.
3. No repositório, vá em **Settings → Pages**.
4. Em **Source**, escolha **Deploy from a branch**, branch **main**, pasta **/ (root)** e salve.
5. Aguarde 1–2 minutos. O app estará em: `https://SEU-USUARIO.github.io/agenda-lene/`

## 📲 Instalar no celular (como aplicativo)

- **Android (Chrome):** abra o link → menu ⋮ → **Adicionar à tela inicial** / **Instalar app**.
- **iPhone (Safari):** abra o link → botão Compartilhar → **Adicionar à Tela de Início**.

Depois de instalado, funciona rápido e até sem internet.

## 💾 Onde ficam os dados?

Os dados ficam salvos **no próprio aparelho** (navegador). Não há servidor — ninguém mais tem acesso.

Importante:
- Use sempre o **mesmo navegador/aparelho** para gerenciar a agenda.
- Faça **backup regularmente** em ⚙ Configurações → Backup completo (gera um arquivo .json). Guarde no Google Drive ou envie para si mesma no WhatsApp.
- Para trocar de celular, use **Restaurar backup** no aparelho novo.
- Não limpe os dados de navegação do site, ou a agenda será apagada.

## 🔒 PIN

Defina um PIN em ⚙ Configurações → Segurança para proteger o acesso ao app.

## 🔔 Lembretes

Ative em ⚙ Configurações → Lembretes. Com o app aberto (ou instalado, no Android), você recebe alertas 1h, 30 min e 15 min antes de cada atendimento.
